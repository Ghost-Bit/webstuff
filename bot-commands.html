PUYO ROBOT COMMAND LIST:
p?about - Sends some basic information about the bot when triggered.
p?roll [INT] ([STR]) - Rolls a dice with the amount of sides inputted.
p?s-roll [INT] - Same as p?roll, but command output is sent directly to it's status rather than sent as a message in chat,
p?vs-roll [INT 1] [INT 2] - Rolls two dice, then tells the difference of said dice.
p?help - Sends a link to this page.
p?wow [UPPERCASE ONLY STRING (NO NUMBERS)] - Sends emotes in chat replacing what it received
p?st-roll [INT] - Sets status to output of dice roll with no leading text.
p?userphone [SERVER ShortName] [Message Content] - Sends a message to a server's specified Bot or Userphone channel.
p?servers-userphone - Sends a list in the chat of registered Server Shortnames for usage with the Userphone Command.
p?leaderboard [PAGE NUMBER (Optional, but will only send first page unless provided)] - Gets a page from the cross-server leaderboard, containing all user messages counted since added to a server (Does not count messages before added or while offline!)
p?die - ???
p?cool - Basically a randomized True/False thing.
p?r-cool Same as p?cool, but reacts instead of sending as a gif.
p?rate [thing1] [thing2] - Outputs something's something level.
p?offtopic [Page #] - Grabs a page from AiDungeon's offtopic message log. Fun to be had here!
p?box [width] [height] - outputs an ascii box based on the variables provided.
p?1~box [width [height] - outputs a block-style box based on the variables provided.
p?suggest [message content] - Sends a suggestion to the dev's suggestions folder.
p?whoami - Sends your user tag in chat as text
